# Design & Topology Reference

## UI Design Library (ARCH_DOCS/04-DESIGN/)

**Key File: DESIGN_SYSTEM.md (v4.0)**

### Principles

- **Invisible Design**: Best interface is no interface.
- **Trust Through Restraint**: Professional, simple for SMBs.
- **One Primary Action**: Per screen.
- **Data Over Decoration**: No emojis, Lucide icons only when meaningful.
- **White Space = Confidence**.

### Color System

- **Primary**: Navy #1E3A5F (headers/buttons).
- **Semantic**: Success #059669, Error #DC2626, etc.
- **Neutral**: Gray scale (gray-900 #111827 max contrast).

### Typography

- Font: Inter (sans), JetBrains Mono (mono).
- Scale: display 30px/600, body 14px/400.

### Spacing

4px scale: 4px (1), 8px (2), ..., 64px (16).

### Components

- Buttons: 40px h, 6px radius, 500 weight.
- Cards: white bg, 1px gray-200 border, 8px radius.
- Tables: gray-50 headers, hover gray-50.
